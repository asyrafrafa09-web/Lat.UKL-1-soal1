# Program untuk menghitung biaya pengiriman paket berdasarkan berat,jarak,dan volume barang
Langkah kerjanya:
1. Pengguna memasukkan berat (kg), jarak (km), serta panjang, lebar, dan tinggi (cm) paket.
2. Program menghitung volume = panjang × lebar × tinggi.
3. Menentukan biaya per kg:
-Jika jarak ≤ 10 km → Rp 4.250/kg
-Jika jarak > 10 km → Rp 6.000/kg
5. Jika volume > 100 cm³, ditambah biaya Rp 50.000.
6. Total biaya = (berat × biaya per kg) + biaya tambahan volume.
7. Hasil perhitungan ditampilkan sebagai rincian biaya pengiriman.
